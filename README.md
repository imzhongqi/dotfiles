# my dotfiles

## Requirements

- stow
- zsh
- curl
- git
- unzip

```bash
./init
```

## Init Submodule
```bash
git submodule init # init the submodule
git submodule update
```

## Install
```bash
stow vim zsh # etc...
```

## Uninstall
```bash
stow -D vim
```
